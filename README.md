## Main Features
- Display list of hotels from the backend data
- Sort the list by price


## Get the source code

```shell
git clone https://github.com/remoteriver/Qantas-Hotels
```

## Install & Start

1. Install packages:

```shell
npm install
```

2. Start the APP

```shell
npm start
```

3. Check http://localhost:3000 in browser

## Test

```shell
npm test
```
